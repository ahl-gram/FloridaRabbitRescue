# Florida Rabbit Rescue Directory

A free, independently maintained directory of verified rabbit rescue organizations in Florida.

## What This Is

This site helps Florida residents find reputable rabbit rescue organizations. Each listing includes mission statements, founding dates, 501(c)(3) status, and board transparency details — all sourced from publicly available records.

The directory is not affiliated with any rescue organization, House Rabbit Society, or Rabbit.org Foundation.

## What's on the Site

- **Directory** — 15 rabbit rescue organizations with verified public records
- **Shared Values** — Core principles of responsible rabbit rescue, drawn from House Rabbit Society and Rabbit.org Foundation
- **Before You Adopt** — Honest guidance for prospective rabbit owners
- **About** — How organizations are selected and verified
- **Contact** — Submit corrections or suggest new organizations

## Contributing

If you represent a rescue organization listed here and need to update your information, or if you believe an organization is missing, please use the contact form on the site.
